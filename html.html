O que é HTML

HTML ( HyperText Markup Language ) é uma linguagem de marcação cuja função é organizar e definir a estrutura e os conteúdos de páginas web. Ela foi criada em meados de 1990, ela utiliza tags para definir elementos essenciais de uma página ou site, tais quais títulos, parágrafos, imagens, links, mídias e diversos outros componentes relacionados.
