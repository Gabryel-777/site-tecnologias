Java Script

O que é Java Script

JavaScript é uma linguagem de programção de alto nível e bem avançada, a qual é frequentemente utilizada tanto no backend quanto no frontend, estabelecendo essa conexão com o cliente e com o servidor. Ela foi criada em meados de 1995 justamente com o objetivo de adicionar interação e um toque de dinamismo ás páginas web.
